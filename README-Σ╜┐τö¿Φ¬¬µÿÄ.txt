Flashcardo（手機優化版）

這個版本特別針對手機使用做了調整，適合 flashcard / quiz 場景。

新增的手機友好功能：
1. 底部固定操作列：上一張 / 顯示或隱藏圖片 / 下一張
2. 題目卡可直接點一下顯示圖片
3. 圖片區左右滑動切換卡片
4. 手機上卡片列表改成抽屜式面板，比較不擋畫面
5. 進度條與卡片計數更清楚
6. 雙擊圖片可快速放大 / 還原
7. 搜尋列可一鍵清除
8. 狀態會自動記住，下次打開可接著用

最直接的用法：
A. 電腦上解壓後，整個資料夾上傳到任何靜態網站服務（例如 GitHub Pages / Netlify / Vercel）
B. 用手機打開那個 HTTPS 網址
C. iPhone：Safari 分享 -> 加入主畫面
D. Android：Chrome 選單 -> 安裝 App / 加到主畫面

補充：
- 這個版本仍然是 PWA，不是原生 .ipa 或 .apk。
- 但在手機上加到主畫面後，使用方式已經很接近 App。
- 若你每天往 images/ 裡加圖片，GitHub Actions 版本可自動更新 cards.json 與快取版本。
